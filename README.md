# TubeGAI

TubeGAI는 유튜버와 콘텐츠 크리에이터를 위한 비디오 제작 파이프라인을 자동화하고 간소화하도록 설계된 **AI 기반 통합 크리에이터 워크플로우 솔루션**입니다. 생성형 AI와 YouTube 데이터를 결합하여, TubeGAI는 트렌드 발견, 아이디어 생성, 프로젝트 관리 및 효율적인 콘텐츠 제작을 돕습니다.

## 🚀 주요 기능

### **1. 데이터 기반 대시보드**

- **트렌드 분석**: YouTube의 실시간 트렌드 토픽을 발견하여 다음 비디오에 대한 영감을 얻으세요.
- **성과 추적**: 인터랙티브 차트를 통해 채널 성장(조회수, 구독자) 및 콘텐츠 성과 지표를 시각화합니다.
- **통합 워크플로우**: 단일 인터페이스에서 프로젝트, 채널 및 라벨을 통합 관리합니다.

### **2. 프로젝트 관리**

- **프로젝트 목록**: 비디오 프로젝트를 상태별(아이디어, 스크립트 작성, 편집, 완료)로 체계적으로 정리합니다.
- **검색 및 필터**: 이름이나 테마로 프로젝트를 빠르게 찾을 수 있습니다.
- **테마 선택**: 트렌딩 테마에서 직접 새 프로젝트를 시작하세요.

### **3. 스튜디오 (Coming Soon)**

제작의 모든 단계를 처리하는 포괄적인 도구 모음:

- **스크립트 생성**: AI 기반 스크립트 작성.
- **스토리보드**: 촬영 전 장면 시각화.
- **자산 관리**: 환경, B-Roll 및 장면 관리.
- **포스트 프로덕션**: 자막, 컬러링, 썸네일 및 SEO 최적화.
- **내보내기**: YouTube용 원클릭 최적화.

### **4. 현대적인 UI/UX**

- **반응형 디자인**: 데스크톱 및 모바일에 완벽하게 최적화되었습니다.
- **Shadcn UI**: 아름답고 접근성 높으며 커스터마이징 가능한 컴포넌트.
- **다크 모드 지원**: 크리에이터를 위해 설계된 세련된 인터페이스.

## 🛠 Tech Stack

- **Framework**: [React Router v7](https://reactrouter.com/) (formerly Remix features)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Charts**: [Recharts](https://recharts.org/)

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1.  Clone the repository:

    ```bash
    git clone https://github.com/sungalex/tubegai.git
    cd tubegai
    ```

2.  Install dependencies:

    ```bash
    npm install
    ```

3.  Start the development server:

    ```bash
    npm run dev
    ```

4.  Open `http://localhost:3000` in your browser.

## 🏗 Building for Production

To create a production build:

```bash
npm run build
```

The output will be in the `build/` directory, ready for deployment.
